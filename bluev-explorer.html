About BlueV Explorer

BlueV Explorer is designed to be the next generation .rpf file editor for GTA V modders. Built from the ground up with modern modding needs in mind, it provides a seamless experience for both beginners and advanced users.

Features

  • Full support for GTA V Enhanced Edition
  • Read and modify .rpf archive files
  • Intuitive drag-and-drop interface
  • Advanced search and filtering
  • Backup and restore functionality

Why BlueV Explorer?

The GTA V modding community has long needed a modern tool that supports the Enhanced Edition. BlueV Explorer aims to fill this gap while providing a better user experience than existing tools.

Development Status

BlueV Explorer is in the final stages of development. The core functionality is complete, and only file modification features remain. Stay tuned for the official release!
