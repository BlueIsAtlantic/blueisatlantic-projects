About TornadoV

TornadoV started as an update to CamxxCore's "Tornado Script" to make it stable with newer versions of GTA V. It evolved into a comprehensive tornado simulation mod that brings realistic natural disasters to San Andreas.

Features

  • Tornado physics and behavior
  • Destructive environmental interactions
  • Customizable tornado strength and appearance
  • Vehicle and object pickup mechanics
  • Compatible with latest GTA V versions

Installation For TornadoV

  • Install ScriptHookV and ScriptHookVDotNet
  • Download TornadoV .zip package from Gofile.io
  • Extract the files to your GTA V scripts folder except the README file
  • Launch the game and enjoy!
  • If you encounter crashes on tornado's spawn, try to install Menyoo.

Installation For TornadoV++

  • Install ScriptHookV
  • Download TornadoV++ .zip package and extract it
  • Place ONLY .asi file in your GTA V directory (where GTA5.exe is located)
  • Launch the game and enjoy TornadoV++!
